### 18 апреля 2026 — День открытых дверей: публичная презентация платформы

Команда проекта представила платформу на Дне открытых дверей Московского Политеха. Стенд посетили абитуриенты, их родители и преподаватели. Живая демонстрация включала полный сценарий: регистрацию студента, подачу заявки на консультацию и просмотр статуса в личном кабинете.

Особый интерес вызвали тёмная тема и модальное окно записи — компоненты, которые разработала наша тройка. По итогам дня собрано более 30 положительных отзывов.

## История проекта (до нашего участия)

### 05 марта 2025 — Первый стабильный деплой платформы на тестовый сервер

Бэкенд-команда успешно задеплоила рабочую версию API на тестовый сервер. Docker Compose объединяет три контейнера: FastAPI-приложение, PostgreSQL и Nginx в роли reverse proxy. Все endpoint'ы аутентификации и заявок доступны по HTTPS.

Фронтенд получил стабильный адрес тестового сервера и начал полноценную интеграцию — отправку реальных запросов вместо заглушек.

### 28 февраля 2025 — Автоматизированное тестирование бэкенда на Pytest

Команда тестирования настроила инфраструктуру для автоматизированных тестов. Покрыты ключевые маршруты: регистрация, вход, создание заявки, смена статуса. В GitHub Actions добавлен шаг автозапуска тестов при каждом pull request.

### 10 февраля 2025 — Архитектура бэкенда и жизненный цикл заявок

Бэкенд-отдел утвердил схему жизненного цикла заявки: «Новая» → «Рассматривается» → «Подтверждена» → «Завершена» / «Отменена». На каждом переходе генерируется email-уведомление студенту.

### 14 января 2025 — Завершение разработки ядра REST API

Бэкенд-команда завершила реализацию основных endpoint'ов: регистрация и вход (JWT), создание и обновление статуса заявки, расписание психолога. Настроены Alembic-миграции и система email-уведомлений.

### 08 декабря 2024 — Передача макетов Figma фронтенду

Дизайн-отдел завершил полный комплект макетов: главная страница, новости, формы регистрации/входа, личные кабинеты студента и психолога — в светлой и тёмной теме, на трёх брейкпоинтах. Фронтенд-отдел провёл разбор макетов и согласовал финальные токены дизайн-системы.

### 20 ноября 2024 — Анализ аналогов и выбор технологического стека

Команда провела анализ существующих решений психологической поддержки студентов в российских университетах (МГУ, СПбГУ). По итогам утверждён стек: Python + FastAPI, React + TypeScript, PostgreSQL, Docker.
