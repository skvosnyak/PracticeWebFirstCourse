Участники проекта

Три студента группы 251-321, присоединившихся к фронтенд-команде проекта в 2026 году. Рассказываем о своём конкретном вкладе.

ЖА
Жемчугов Алексей
Frontend · Header & News
  • Разработал React-компонент шапки (Navbar): логотип, навигационное меню с React Router, адаптивный бургер с CSS-анимацией и управлением фокусом
  • Реализовал страницу новостей: карточки на TypeScript-интерфейсах, серверная пагинация и фильтрация по тегам через TanStack Query
  • Настроил конфигурацию проекта: Vite, пути TypeScript, ESLint, Prettier и .editorconfig
  • Участвовал в разработке личного кабинета психолога: список заявок, карточка клиента
ЗД
Зарубин Дмитрий
Frontend · Adaptive & Modal
  • Обеспечил адаптивность всех React-компонентов для 5 брейкпоинтов: от 320 px до широких десктопных экранов
  • Разработал React-компонент модального окна записи на приём: React Hook Form + Zod для валидации, управление фокусом (focus trap) и закрытие по Escape
  • Реализовал CSS-анимации, плавные переходы и оверлеи для интерактивных компонентов
  • Участвовал в разработке личного кабинета психолога: форма обратной связи, статусные бейджи
ЯЕ
Якунин Егор
Frontend · Dark Theme
  • Разработал систему тем целиком на CSS-переменных без дублирования правил — переключение темы не перегружает браузер
  • Реализовал переключатель тем: хранение выбора пользователя через Zustand (синхронизация с localStorage), поддержка prefers-color-scheme
  • Обеспечил соответствие контраста всех текстовых элементов стандарту WCAG 2.1 AA в обоих темах
  • Участвовал в разработке личного кабинета психолога: тёмное оформление таблицы расписания

Наше место в проекте

Мы трое — небольшая часть большой команды. В фронтенд-отделе работают и другие разработчики, которые трудятся над проектом с самого начала. Помимо фронтенда, в проекте участвует ещё несколько отделов:

  • Бэкенд FastAPI, PostgreSQL, REST API, деплой на сервер
  • Дизайн Figma-прототипы, UX-исследования, дизайн-система
  • Тестирование Ручное и автоматизированное QA, регрессионное тестирование
  • Менеджмент Agile-процессы, спринты, коммуникация с заказчиком
